
An AI-powered Retrieval-Augmented Generation (RAG) chatbot that allows users to query multiple documents (PDF + TXT) and receive accurate, context-aware answers using Large Language Models.

Features-
Semantic search over multiple documents (PDF, TXT)
Context-aware answers using LLM (GPT-based)
Multi-file document ingestion
Fast similarity search with FAISS (vector database)
Chat-style interface using Streamlit
Source-aware responses
Document chunking for improved retrieval

working-
End-to-end RAG pipeline integrating:
Document ingestion → Chunking → Embeddings → Retrieval → LLM response
Semantic similarity search using vector embeddings
Modular and scalable architecture for multi-file support
Traceable outputs with document-level source references
